# Login linux via SSH

- ### start `Linux` in QEMU
  ```bash
  $ cd linux_learning_environment
  $ ./script/qemu_linux_start
  ```

- ### login `Linux`, run:
  ```bash
  $ cd linux_learning_environment
  $ ./script/login_linux
  # uname -a
  Linux buildroot 4.16.7 #1 SMP Sat Jan 19 21:50:29 CST 2019 armv7l GNU/Linux
  ```
